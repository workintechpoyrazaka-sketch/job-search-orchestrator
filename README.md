# Job Search Orchestrator

An AI-native job-search system: collect remote postings, score and
filter them, and draft tailored applications — with a human in the loop.

Single orchestration core, many source adapters (Remotive now,
Himalayas / WWR next). SQLite as system of record, DuckDB for analytics.

## Stack
Python, SQLite, DuckDB, Anthropic API
